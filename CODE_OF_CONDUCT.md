# Code of Conduct

## Our Pledge

Youp Grid is an open-source project. We are committed to making participation in this project respectful, constructive, and welcoming for everyone.

## Expected Behavior

- Be respectful and professional.
- Focus feedback on ideas, code, APIs, documentation, and user impact.
- Assume good intent when discussing bugs, tradeoffs, and design choices.
- Respect different backgrounds, experience levels, and constraints.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks.
- Sexualized language, imagery, or attention.
- Threats, doxxing, or sharing private information without consent.
- Trolling, sustained disruption, or bad-faith participation.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, and other contributions that violate this code. Maintainers may also temporarily or permanently restrict participation in project spaces.

Report concerns to the maintainer through GitHub by opening a minimal issue that asks for a private contact channel. Do not include sensitive personal details in a public report.

## Scope

This code applies to GitHub issues, pull requests, discussions, and any public community spaces managed by the Youp Grid project.
